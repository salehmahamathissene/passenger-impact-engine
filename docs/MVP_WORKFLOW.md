# PIE MVP Workflow (Narrow Scope)

## MVP: EU261 Exposure Forecast
### Input
- passenger count
- delay distribution or probability
- disruption type (delay/cancel/denied boarding)
- itinerary context (connections optional)

### Output
- expected payout
- percentile outcomes (P50 / P95)
- payout histogram / summary table

## Success metric (what we will measure)
PIE produces a consistent, repeatable estimate with confidence bounds.
Next iteration adds scenario comparison: “Option A vs Option B savings.”
